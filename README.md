# APEX CAPITAL — Investment Growth Calculator

Single HTML file. Open in browser. No install, no login, no data sent anywhere.

---

## Inputs

| Field | What it is |
|---|---|
| Initial Capital | Lump sum you start with |
| Annual Interest Rate | Expected yearly return (%) |
| Investment Duration | Time horizon in years (1–100) |
| Compounding Frequency | How often interest compounds — Annual / Semi-Annual / Quarterly / Monthly / Daily |
| Contribution Amount | Fixed recurring top-up (optional) |
| Contribution Frequency | Monthly or yearly |
| Inflation Rate | Annual inflation % to apply (optional, toggle on/off) |

---

## Output

Hit **Calculate Growth**. You get:

- **4 summary numbers** — Final value, inflation-adjusted value, total contributed, total interest earned
- **Line chart** — Portfolio value year by year (+ inflation-adjusted line if enabled)
- **Bar chart** — Your contributions vs. interest earned, cumulative per year
- **Table** — Starting balance, contributions, interest, and ending balance for every year

---

All calculations run locally. Fields validate before calculating and highlight red if out of range.
